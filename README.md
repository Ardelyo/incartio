# FINCARS - Jelajahi Sejarah Pasar

Game finansial mobile-first buatan anak bangsa, tempat grafik nilai tukar menjadi lintasan yang kamu lewati.

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/f3c749fb-c6fd-43fc-b018-21538d2afa2e

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Copy [.env.example](.env.example) to `.env` and set any server-side market data keys you want to use.
3. Start the secure API proxy:
   `npm run server`
4. In another terminal, run the app:
   `npm run dev`
